# AgentX — Research Agent

A research agent built with LangGraph, equipped with web search and Wikipedia tools. It can browse the web, read encyclopedic knowledge, and maintain memory across conversational turns.

## Features
- **DuckDuckGo Search**: Searches the web for real-time information.
- **Wikipedia**: Looks up encyclopedic and historical information.
- **Memory**: Remembers context across conversation turns.
- **Reasoning Trace**: View the steps the agent took to arrive at its answer in the Gradio UI.

## Setup
1. Clone the repository
2. Install requirements: `pip install -r requirements.txt`
3. Rename `.env.example` to `.env` and add your Groq API Key.
4. Run the app: `python app.py`
